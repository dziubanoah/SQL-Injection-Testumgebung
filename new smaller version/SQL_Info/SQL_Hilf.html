Wie Sieht die SQL Syntax aus ?

Was ist SQL ?

SQL ermöglicht das Abfragen von Daten in Datenbanken.

Verwendet zur Überprüfung von Passwörtern und Nutzerdaten.

Sichert den Abgleich von eingegebenen und gespeicherten Daten.

Ermöglicht den Zugriff nur bei korrekten Eingaben.





Was ist eine SQL Injection?

Eine **SQL-Injection** tritt auf, wenn das Backend unsicher ist.

Alle SQL-Anfragen werden direkt an die Datenbank weitergeleitet.

Angreifer können schadhafte SQL-Befehle in Eingabefelder einfügen.

Dadurch kann unbefugter Zugriff auf Daten erfolgen.

OR Statement in SQL

dadurch, dass SQL Syntax in die Datenbank kommt, ist es möglich

statt eines Passworts der Datenbank eine Option zu geben.

Mit einem OR statement in SQL kann man einen eingegebenen wert

eine Alternative geben. z.B passwort OR 1=1. Hier wird SQL

mit OR 1=1 eine Alternative zum eigentlichen passwort gegeben.

Wie interpretiert SQL ein Passwort ?

Eine Datenbank würd in unsicheren Backend systemen das Passwort

idr. als unverschlüsselten String (Zeichenkette) Interpretieren.

Dadurch das OR aber SQL syntax ist darf es nicht im String stehen.

um ein String als verarbeitung zu umgehen wird ' benutzt.

bsp. Pw' OR 1=1. Was die Datenbank jetzt versteht: 'Pw' OR 1=1

Das " schließt den string und erlaubt so SQL syntax.

SQL code auskommentieren

Beim auskommentieren wird alles hinter einem -- für die

Datenbank unsichtbar. so kann z.B alles hinter einem OR 1=1

für die Datenbank unsichtbar gemacht werde. Das ist wichtig, um

eventuelle weitere ' ungültig zu machen, oder um weitere syntax

unsichtbar zu machen. z.B währe so eine eingabe: Pw' OR 1=1 --
was die datenbank sieht: 'Pw' OR 1=1 -- weiterer code / semikolon

Weitere Infos

Hinter Jeder SQL Abfrage muss ein Semikolon, damit die Abfrage

korrekt abgeschlossen wird. Beim auskommentieren wird dies

entfernt, desswegen muss es manuell hinzugefügt werden z.B:

Pw' OR 1=1; -- Was die Datenbank sieht: 'Pw' OR 1=1; -- weiteres;

!WICHTIG! vor und hinter dem -- muss jeweils 1 Leerzeichen sein.

Bring die Codeschnipsel in die richtige Reinfolge.
