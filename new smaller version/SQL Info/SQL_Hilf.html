Was ist SQL ?

SQL ist eine Sprache, mit der man Daten in einer Datenbank abfragen kann, um zum Beispiel ein Passwort zu überprüfen. Damit kann man sicherstellen, dass die eingegebenen Daten mit den gespeicherten übereinstimmen. SQL wird verwendet, um nach bestimmten Informationen zu suchen, wie einem Benutzernamen und Passwort, und die Eingaben zu vergleichen. So wird überprüft, ob die Daten korrekt sind und der Benutzer Zugang bekommt.
