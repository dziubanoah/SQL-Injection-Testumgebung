Datenbank Admin Bereich


Alle Inputs werden als String interpretiert. Damit werte nicht als String interpretiert werden
können " Benutzt werden. Beispiel: Interpretiert wird: "Wert"
Interpretiert wird: "" + Wert + "" im 2. Beispliel wird der wert als nicht string angesehen
Dabei kann es sich z.B um ein SQL Statement handeln. So kann ein Richtiges SQL statement aussehen:
SELECT * FROM persons WHERE PersonID = "Wert" AND PW = "Wert";
Dabei steht "Wert" jeweils für das Passwort und die ID
